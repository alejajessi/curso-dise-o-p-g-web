Etiquetas HTML

este ejemplo muestra como combinar algunas de las etiquetas más básicad de HTML5.
Recuerde que es más importante entender la diferencia entre ellas

Etiqueta ul+1l

si hacemos una lista de mercado dónde el orden no es importante, debemos usar UL

En el caso de que estemos listando elementos dónde el orden es más importante,
como por ejemplo la clasificación mundial de tenistas, debemos usar ol
